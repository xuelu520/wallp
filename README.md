
#哇扑-一个有趣的网站

    <哇扑>是 wall paper 简称，一个用户分享分享图片，分享壁纸的站点。

##环境配置
>php5.5 + mysql + redis  
>php框架采用 codeigniter3  
>壁纸存储方案采用又拍云-云存储，本地数据库只存图片链接。
>前端登录采用QQ登录以及微信登录，暂时不准备用自己的注册系统。
